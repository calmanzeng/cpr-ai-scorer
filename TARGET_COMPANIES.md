# 目标企业分析：执医24项 AI 评分赛道

## 核心发现

所有传统医学教育企业在 GitHub 上的技术足迹 = 零。
这意味着一件事：你要么被当作"稀缺的 AI 能力"高价引入，要么被当作"不懂医疗的外行"拒之门外。
决定因素：你用什么方式进入他们的视野。

---

## Tier 1: 最推荐的 2 家目标

### 🥇 天津天堰科技 (300314.SZ)

| 维度 | 分析 |
|------|------|
| **体量** | A股上市，市值约 40-60 亿 |
| **核心业务** | 医学模拟教学设备（模拟人、VR手术、OSCE考站） |
| **医院覆盖** | 全国 1000+ 医院的技能培训中心 |
| **AI 现状** | 有 VR/AR 产品，但核心评分仍靠人工 |
| **为什么是最佳选择** | (1) 上市有钱投AI (2) 产品线天然需要AI评分 (3) 渠道已有 (4) 品牌认知强 |
| **切入策略** | "你们的 OSCE 考站硬件卖 50 万/套，加上 AI 自动评分模块，可以卖 120 万/套，并且每年收 20 万服务费。这是Demo。" |
| **接触渠道** | 脉脉搜索"天堰"→找产品总监/CTO；天津本地→直接约咖啡 |
| **风险** | 天津国企风格，决策慢；你非医学背景，可能被质疑 |

### 🥈 北京医模科技 (300331.SZ)  /  杭州医惠科技 (300078.SZ)

| 维度 | 分析 |
|------|------|
| **体量** | A股上市，市值约 20-40 亿 |
| **核心业务** | 高端医学模拟人（进口替代）+ 数字化教学系统 |
| **AI 现状** | 几乎为零——连 GitHub 账号都没有 |
| **为什么值得关注** | 硬件制造能力极强（高端模拟人单价 100-200 万），差的就是 AI 软件层 |
| **切入策略** | "你们的高端模拟人可以模拟生理反应，但不能自动评估学生操作。加上AI评分，硬件+软件的 combo 没有竞争对手能做。" |

## Tier 2: 次优但可行

### 🥉 医视界 / 华医网

| 维度 | 分析 |
|------|------|
| **业务** | 住培管理系统 + 在线课程（纯软件，无硬件） |
| **AI 现状** | 只做"在线课程播放"，不做 AI 评分 |
| **优势** | 已经跑通了医院采购流程，有现成的合同模板和招标渠道 |
| **劣势** | 缺乏硬件基因——你需要说服他们"软件还不够，要加 AI" |
| **切入策略** | "你们的住培管理系统卖了 500 家医院，但都是管理功能。加一个 AI 技能评分模块，从每家医院每年多收 10 万。" |

### 科大讯飞 (002230.SZ)

| 维度 | 分析 |
|------|------|
| **业务** | AI 第一大厂，医疗是重点赛道 |
| **AI 现状** | 医疗语音录入强，医学教育有布局但偏语音 |
| **优势** | AI 能力强、品牌强、不差钱 |
| **劣势** | 体量太大——你不是"带方案加入"，是"小兵入场" |
| **不建议作为第一目标** | 但可以作为 Plan B——"讯飞智慧医疗教育"事业部 |

## Tier 3: 黑马——值得跟踪但不主动接触

### 医微讯（VR 手术模拟）

- 有 VR 技术基因、融资到 B 轮
- 体量小→你加入的影响力更大
- 但渠道覆盖太窄

### 医脉通（医生社区）

- 海量医生用户但教育板块弱
- 收购小团队的可能性存在
- 更适合做 C 端执医培训而非 B 端医院
